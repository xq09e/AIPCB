# NEVERLOSE CB CORE PROJECT V9.0 - ПОЛНАЯ КОПИЯ CS2

## Описание
Этот скрипт является ПОЛНОЙ КОПИЕЙ функционала и эстетики читов NeverLose CS2, адаптированных для Combat Warriors. Включает Skeet-стиль ESP, Watermark с датой/временем, PSilent, RCS, Bhop и Skinchanger. Все предыдущие ошибки (V7.1, V8.0) устранены.

## Структура Проекта (GITHUB MASTER V9.0)
- `README.md`: Этот файл.
- `index.lua`: **ФИНАЛЬНЫЙ ЗАГРУЗЧИК**. Запускать только его.

## Инструкция по Запуску
1. Скопируйте **весь** код из файла `index.lua` (см. ниже).
2. Вставьте его в консоль исполнителя.
3. Нажмите «Выполнить».

**Горячая клавиша меню:** DELETE
**Watermark:** Включен по умолчанию.
